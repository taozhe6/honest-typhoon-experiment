# ADR-001: 多状态马尔可夫物理核心

状态：Rejected，2026-07-11。

## 背景

旧模型以单一风速状态和机构端点插值为核心，强迫场、变量口径和参数识别均出现结构性风险。新模型需要直接回答三个审计问题：状态向量、全局自由参数数量、密封证伪数据。

## 决策

采用马尔可夫切换状态空间：

- 连续状态 `X=(V,m,Pc,RMW)`。
- 隐状态 `Z={减弱,准稳态,增强}`。
- `V-m` 采用 FAST 耦合 ODE。
- `Pc` 采用 Sparks--Toumi 质量通量方程。
- `RMW` 采用绝对角动量零阶闭合。
- 转移核仅开放 `persistence_logit` 与 `regime_width` 两个统计参数。
- 三个评分字段为 `V、Pc、RMW`，通道依赖性单独审计。
- 外部强迫全部携带有效时刻、变量定义和数据来源。

## 结果

- 风切、内核水汽、海洋冷尾流、中心气压和内核尺度均进入状态演化。
- 隐状态只进入中心气压方程，对风速趋势的反事实效应为 `0.0 m/s/day`。
- JTWC 西北太平洋 `V/Pc/RMW` 完整样本的参与率有效维数为 `1.4641`，显示三字段高度冗余。
- 当前风暴只提供初值与强迫，参数来自历史训练产物。
- RMW 角动量闭合、固定柱流速和三态中心成为首批可推翻结构假设。
- 实时资格取决于 2022--2024 密封回报检验。

零风速效应已经触发结构淘汰。正式可识别性还需要观测误差协方差、白化灵敏度/Fisher 信息和剖面似然审计；该 ADR 保留为失败记录。下一版需要双风环状态、对应观测算子和重新登记的参数预算。

## 延后方案

- 标量 SDE 保留为数据驱动基线。
- 完整耦合数值天气模式保留为业务指导比较对象。
- 机构强度端点保留为外部评估数据。
- RMW 的额外经验修正等待 v0 回报误差提供证据。
