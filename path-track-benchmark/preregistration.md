# A 支线路径回报预注册

冻结日期：2026-07-15  
状态：`preregistered-before-results`  
结果盲态：冻结前只检查文件目录、字段格式、TECH 定义和代表性文件中的模式覆盖；尚未把任何预报位置与验证位置配对，也未计算路径误差。

## 1. 研究问题与交付

在西北太平洋 2022--2024 年样本上：

1. 复现加拿大模式 `CMC` 与 NAVGEM/NOGAPS `NGX` 的逐起报路径。
2. 构造本项目的两模式等权球面共识 `DYC2`。
3. 在完全相同的风暴、起报时刻、提前量和验证位置上报告三者误差及台风聚类区间。
4. 报告 `DYC2` 的逐风暴交叉拟合误差半径与实际覆盖率。
5. 明确资料时效、缺测、验证路径不确定性和结论边界。

## 2. 三个永久审计问题

### 2.1 状态向量

路径输出在每个有效时刻的状态为

\[
X_t=(\phi_t,\lambda_t),
\]

其中 `phi` 为纬度，`lambda` 为经度。起报时刻、提前量、模式 TECH、风暴 ID 和资料抓取时刻属于强制元数据。`DYC2` 是逐提前量诊断共识，不引入未观测的动力状态。

### 2.2 参数与独立输入

- 拟合自由参数：0。
- 固定权重：`CMC=0.5`、`NGX=0.5`，不按本样本结果调整。
- 路径输入：两个不同全球动力模式的预报中心位置。两者可能共享资料同化和追踪算法，因此不宣称统计独立；其误差相关性将作为测量结果报告。
- 验证通道：JTWC/USA 事后最佳路径中心位置。该通道是事后专家分析估计，缺少逐点独立位置真值和观测误差方差。
- IBTrACS RMW 完全排除；本项目只读取 `USA_ATCF_ID/ISO_TIME/USA_LAT/USA_LON` 及身份元数据。

### 2.3 证伪数据

每条预报由同一风暴、同一起报时刻和同一有效时刻的事后最佳路径位置证伪。主要误差为 WGS84 椭球上的中心位置测地距离。解析冲突、未来资料泄漏、样本不足和无法复现的重复行均触发显式失败。

## 3. 数据源与来源纪律

### 3.1 预报路径

- 来源：[UCAR/NCAR Tropical Cyclone Guidance Project Global Repository](https://hurricanes.ral.ucar.edu/repository/)。
- 文件：`adecks_open/{year}/awpNNYYYY.dat`。
- 格式：[ATCF a/b-deck](https://hurricanes.ral.ucar.edu/realtime/index.php#about_adecks)。
- 模式一：`CMC`，NHC TECH 表定义为 Canadian model。
- 模式二：`NGX`，NHC TECH 表定义为 NAVGEM/NOGAPS with GFS tracker。
- 两者属于 ATCF `late`/原始动力模式指导。历史文件缺少逐条真实公开时刻，本研究称其为“模式循环预报”，不声称在循环时刻已经可供预报员使用。

### 3.2 验证路径

- 来源：[NOAA/NCEI IBTrACS v04r01 Western Pacific CSV](https://www.ncei.noaa.gov/data/international-best-track-archive-for-climate-stewardship-ibtracs/v04r01/access/csv/ibtracs.WP.list.v04r01.csv)。
- 字段：`USA_ATCF_ID`、`ISO_TIME`、`USA_LAT`、`USA_LON`、`SEASON`、`BASIN`、`TRACK_TYPE`。
- 只采用 `USA_LAT/USA_LON` 同时存在的准确有效时刻。
- 下载 URL、HTTP 头、字节数和 SHA-256 必须写入运行清单。

### 3.3 方法来源

- 动力模式共识依据 Goerss (2000), *Tropical Cyclone Track Forecasts Using an Ensemble of Dynamical Models*, DOI [`10.1175/1520-0493(2000)128<1187:TCTFUA>2.0.CO;2`](https://doi.org/10.1175/1520-0493%282000%29128%3C1187%3ATCTFUA%3E2.0.CO%3B2)。
- 测地距离使用 GeographicLib 的 WGS84 inverse geodesic，方法依据 Karney (2013), DOI [`10.1007/s00190-012-0578-z`](https://doi.org/10.1007/s00190-012-0578-z)。

## 4. 冻结样本

- 海盆：`WP`。
- 年份：2022、2023、2024。
- 风暴号：`01`--`49`；`90`--`99` 投资区编号全部排除。
- 起报时刻：a-deck 中 `CMC` 或 `NGX` 实际出现的 UTC 循环。
- 主要提前量：24、48、72、96、120 小时。
- 验证匹配：`valid_time = cycle_time + tau`，要求 IBTrACS 存在完全相同的 `ISO_TIME`；不对最佳路径时间插值。
- 主分析样本：`CMC`、`NGX` 和验证位置三者齐全的 `(storm, cycle, tau)` 配对交集。
- 起报定位要求：同一风暴和起报时刻必须存在一个可解析的模式位置；不使用事后最佳路径替换模式初始位置。

## 5. ATCF 解析与质控

1. 按 ATCF 列读取海盆、风暴号、循环、TECH、提前量、纬度和经度。
2. 经纬度按十分之一度与半球字母解析，经度统一为 `[-180, 180)`。
3. 相同 `(storm, cycle, TECH, tau)` 的风圈重复行若位置一致则去重。
4. 同一键出现不一致位置时，该键进入冲突表并从主分析排除；冲突数量大于 0 时报告失败状态。
5. 纬度超出 `[-90,90]`、经度无法规范化、提前量为负或循环时间非法时直接拒绝该行。
6. 每个源文件记录 URL、下载时间、字节数与 SHA-256。

## 6. 本项目共识路径

把每个模式位置转为三维单位向量：

\[
u(\phi,\lambda)=(\cos\phi\cos\lambda,\cos\phi\sin\lambda,\sin\phi).
\]

冻结的共识为

\[
u_{DYC2}=\frac{u_{CMC}+u_{NGX}}{\lVert u_{CMC}+u_{NGX}\rVert},
\]

再转换回纬经度。向量范数低于 `1e-12` 时判为几何失败。权重、模式和公式在读取误差结果后保持冻结。

## 7. 指标与不确定性

每个模式和每个主要提前量报告：

- 配对记录数、风暴数、每个风暴的记录分布和 Kish 有效风暴数。
- 路径误差均值、中位数、80 分位数，单位 km。
- 以台风为 block 的 2,000 次 bootstrap 95% 区间，随机种子 `20260715`。
- `DYC2-CMC` 与 `DYC2-NGX` 的逐例配对误差差及台风聚类区间。
- `CMC` 与 `NGX` 路径误差的 Pearson 与 Spearman 相关，区间同样按台风 bootstrap。

公开展示四舍五入到整数 km；机器可读输出保留完整精度。统计单位始终是台风，记录数不冒充独立样本数。

## 8. 路径区间

对每个留出风暴和每个提前量，只使用其余风暴的 `DYC2` 误差，估计 50%、80%、95% 径向误差分位数。报告：

- 交叉拟合半径的中位数与台风聚类区间。
- 留出风暴实际误差落入各半径的覆盖率与台风聚类区间。
- 各提前量训练风暴数；少于 10 个训练风暴时该区间标为不可估计。

这些半径是经验误差分布，中心仍为 `DYC2`。它们不代表二维高斯概率，也不把两个动力模式视为独立样本。

## 9. 缺测与敏感性

- 按年份、模式、提前量和风暴报告原始行数、唯一轨迹点数、可验证点数和配对保留率。
- 主结果使用严格双模式配对样本。
- 敏感性一：分别报告各模式全部可验证样本，标注其样本构成不同。
- 敏感性二：主要提前量外，若资料允许，附报 12、36、60、84、108 小时；不用于毕业判定。
- 不填补模式缺测，不外推终止后的路径，不按误差大小删除离群点。

## 10. 预注册判据

A 支线在以下条件全部满足时达到当前项目定义的“毕业”：

1. `CMC` 与 `NGX` 两套模式路径能够从公开归档重新下载和解析。
2. `DYC2` 对每个完整配对案例生成路径位置，并报告自己的误差分布。
3. 24、48、72 小时各至少有 10 个台风和 50 个完整配对案例。
4. 三套路径在完全相同案例上的误差、聚类区间、缺测和来源哈希全部公开。
5. 结论明确区分模式循环时间、真实公开时间与事后最佳路径。

没有预注册的技巧胜负门槛。`DYC2` 若更差，结论直接写“更差”；若区间无法区分，结论写“差异未分辨”；只有配对差的 95% 聚类区间排除 0 时才写方向性差异。

## 11. 会证伪什么

- 两模式覆盖低于第 10 节样本门槛：证伪“当前样本足以毕业”。
- 重复键位置冲突：证伪“归档可无歧义复现”。
- 交叉拟合 80% 半径覆盖明显偏离目标且区间排除 0.80：证伪“当前经验区间已校准”。
- `DYC2` 配对误差更大且区间排除 0：证伪“简单两模式共识在该样本改善路径”的命题。
- 验证位置只能通过模式未来资料或人工结果筛选获得：证伪整个回报设计。

## 12. 偏离纪律

冻结后任何模式替换、年份变化、提前量变化、真值字段变化、去重规则变化、统计量变化或阈值变化，必须先写入 `deviations.md`，标注发生在查看哪些结果之前或之后。确认性结果保持本预注册版本；偏离结果单独标为探索性。
